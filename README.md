# sofia-supermarkets-api
An API to retrieve products information from supermarkets in Sofia

## Tech Stack
* Kotlin
* Gradle  
* Spring Boot

## Running Locally
* **Unix** `./gradlew bootRun`
* **Windows** `gradlew.bat bootRun`

## Supported supermarkets

- [x] Billa
- [x] Fantastico
- [x] Kaufland
- [x] Lidl
- [x] T-Market
- [ ] Kam Market
- [ ] CBA
- [ ] ProMarket
